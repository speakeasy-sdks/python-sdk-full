# CardCardBankName

One of ["Kotak", "ICICI", "RBL", "BOB", "Standard Chartered"]. Card bank name, required for EMI payments. This is the bank user has selected for EMI


## Values

| Name                 | Value                |
| -------------------- | -------------------- |
| `KOTAK`              | Kotak                |
| `ICICI`              | ICICI                |
| `RBL`                | RBL                  |
| `BOB`                | BOB                  |
| `STANDARD_CHARTERED` | Standard Chartered   |
