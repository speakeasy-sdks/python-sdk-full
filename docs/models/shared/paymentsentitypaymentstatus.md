# PaymentsEntityPaymentStatus

The transaction status can be one of  ["SUCCESS", "NOT_ATTEMPTED", "FAILED", "USER_DROPPED", "VOID", "CANCELLED", "PENDING"]


## Values

| Name            | Value           |
| --------------- | --------------- |
| `SUCCESS`       | SUCCESS         |
| `NOT_ATTEMPTED` | NOT_ATTEMPTED   |
| `FAILED`        | FAILED          |
| `USER_DROPPED`  | USER_DROPPED    |
| `VOID`          | VOID            |
| `CANCELLED`     | CANCELLED       |
| `PENDING`       | PENDING         |
